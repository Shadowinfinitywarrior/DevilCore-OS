# DevilCore OS Architecture

## Based on SliTaz GNU/Linux

---

## 1. System Overview

```
┌─────────────────────────────────────────────────────────────────────┐
│                        DEVILCORE OS                                 │
│              "Lightweight. Lethal. Limitless."                      │
├─────────────────────────────────────────────────────────────────────┤
│  BASE: SliTaz GNU/Linux (Rolling Release Fork)                     │
│  KERNEL: Custom Hardened Linux Kernel (DevilKernel)                 │
│  INIT: Custom hybrid init (devil-init) based on BusyBox + OpenRC   │
│  PACKAGE MGR: tazpkg (extended) + devil-pkg (custom overlay)       │
│  FILESYSTEM: ext4 / Btrfs (snapshots) / OverlayFS (live mode)     │
└─────────────────────────────────────────────────────────────────────┘
```

### Kernel Config Highlights
```bash
# DevilKernel .config key options
CONFIG_SECURITY_SELINUX=y
CONFIG_SECURITY_APPARMOR=y
CONFIG_SECURITY_LANDLOCK=y
CONFIG_DEFAULT_SECURITY="apparmor"
CONFIG_BPF_SYSCALL=y
CONFIG_NET_CLS_BPF=y
CONFIG_PACKET=y                    # Raw packet access
CONFIG_CFG80211=m                  # WiFi config
CONFIG_MAC80211=m                  # Monitor mode
CONFIG_USB_MON=m                   # USB monitoring
CONFIG_KVM=y                       # Virtualization
CONFIG_WIREGUARD=y                 # Built-in VPN
CONFIG_PREEMPT=y                   # Low-latency (gaming)
CONFIG_ZRAM=y                      # Compressed RAM
CONFIG_BTRFS_FS=y                  # Snapshots
CONFIG_OVERLAY_FS=y                # Live mode
CONFIG_DRM=y                       # GPU support
CONFIG_SND_HDA_INTEL=m             # Audio
CONFIG_INPUT_JOYSTICK=y            # Gaming controllers
```

### devil-pkg Commands
```bash
devil-pkg install <package>          # Install from devil-repo
devil-pkg suite install cybersec     # Install full cybersec suite
devil-pkg suite install gaming       # Install gaming suite
devil-pkg sandbox install <package>  # Sandboxed install
devil-pkg rollback                   # Btrfs snapshot rollback
devil-pkg build <recipe>             # Compile from source
devil-pkg search <query>             # Search all repos
devil-pkg arsenal list               # List security tools
devil-pkg arsenal update             # Update all sec tools
```
